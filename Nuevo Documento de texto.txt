# CrudUsuarioPHP\_Evidencia

**Evidencia: GA7-220501096-AA5-EV03**

Sistema CRUD de usuarios y gestión de citas, desarrollado con **PHP** y **MySQL**, interfaz responsiva en **HTML + Tailwind CSS + JavaScript (Fetch)**.
Este proyecto permite registrar usuarios, iniciar sesión, asignar o cancelar citas, generar facturas y acceder a módulos personalizados por rol: **cliente**, **empleado** y **administrador**.
Forma parte de la evidencia GA7-AA5-EV03 del proceso formativo del **SENA**.

## 🚀 Características

* Registro de usuarios desde formularios o Postman (API híbrida).
* Inicio de sesión con protección por rol y sesiones PHP.
* Asignación y cancelación de citas desde el frontend o mediante Postman.
* Generación y visualización de facturas asociadas a citas.
* Envio de factura por correo electrónico (PHPMailer).
* Interfaz responsiva con TailwindCSS.
* CRUD completo para administradores (usuarios).
* Separación de módulos por roles (cliente, empleado, administrador).
* Compatible con pruebas en Postman o formularios locales.

## 📖 Uso general del sistema

* Registrar un nuevo usuario desde el formulario o Postman (`POST registrar.php`).
* Iniciar sesión con correo y contraseña.
* El sistema redirige según rol: cliente, empleado o administrador.
* El cliente puede agendar y cancelar citas.
* El empleado puede gestionar citas.
* El administrador gestiona usuarios y visualiza citas.

---

## 🔄 Servicios API implementados

* `POST /registrar.php` - Registro de usuario.
* `POST /login.php` - Inicio de sesión.
* `POST /citas.php` - Asignación de cita.
* `DELETE /citas.php?id=ID` - Cancelación de cita.
* `POST /factura.php` - Generación de factura.
* `GET /listar_facturas.php` - Listado de facturas.
* `GET /listar_citas.php` - Listado de citas (admin).
